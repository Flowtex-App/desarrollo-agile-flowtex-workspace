# Capítulos del informe

Este directorio contiene los capítulos del informe del curso de **Desarrollo Ágil** para el proyecto FLOWTEX.

> Cada capítulo es un archivo `.md` independiente. El orden lo define el prefijo numérico del nombre del archivo. Al ejecutar `./scripts/build-docx.sh` desde la raíz del repositorio, los archivos se concatenan en orden alfabético en un único `.docx`.

## Convención de nombres

```
01-introduccion.md
02-objetivos.md
03-...md
```

## Notas

- Mantener una sola jerarquía de encabezados por archivo: el `# Título` es del capítulo; las secciones internas son `##`, `###`, etc.
- Imágenes en `../assets/` con ruta relativa.
- Estilo y mecánica: ver [`CONVENTIONS.md`](../../CONVENTIONS.md) en la raíz.
